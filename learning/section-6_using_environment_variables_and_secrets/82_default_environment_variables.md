### Notes Section (Main Notes)

**1. Default Environment Variables**
- GitHub Actions also provides a couple of default environment variables that are set automatically: [Default Environment Variables](https://docs.github.com/en/actions/learn-github-actions/environment-variables#default-environment-variables)
- These environment variables can, for example, give you quick access to the repository to which the workflow belongs, the name of the event that triggered the workflow and many other things.
