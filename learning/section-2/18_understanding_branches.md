### Notes Section (Main Notes)

**1. Understanding Git Branches**
- **Create a new branch**:
```cmd
git branch <new-branch-name>
```
- **Switch to the new branch**:
```cmd
git checkout <new-branch-name>
```
- **Merge changes from another branch**:
```cmd
git merge <branch-name>
```
